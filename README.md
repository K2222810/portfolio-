# Portfolio
A personal portfolio website built to showcase my projects, skills, and background in software and game development.

## Description
This portfolio is a front-end project built with React and Vite. It was made to present my work in a clear and modern way, with sections for projects, skills, education, and contact details.

The site includes smooth visuals, a responsive layout, and direct links to my live projects and GitHub work. It also helped me practice React, JSX, component structure, and building a cleaner user experience.

## Planning
Before building, the main focus was deciding what information to include and how to present it clearly.

Planning included:

- Deciding on the main sections of the site
- Choosing which projects to feature
- Organising skills, education, and contact details
- Planning a layout that would work well on different screen sizes
- Thinking about how to make the site feel more modern and interactive

## Brief
The goal of this project was to build a personal portfolio website that could present my work, background, and technical skills in one place.

It needed to be clear, easy to use, and visually strong enough to support job applications and show progress as a developer.

## Build Process
The project started by setting up the app with React and Vite. After that, the main page structure was built with sections for the introduction, skills, projects, and contact information.

Once the layout was working, styling and visual details were added to improve the overall look and feel. Project cards, links, and reusable sections were then added to make the portfolio easier to update and expand later.
